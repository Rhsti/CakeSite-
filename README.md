
# 🍰 Sweet Cake

A responsive cake shop landing page built with HTML, CSS, and vanilla JavaScript.

## Features

- 📱 Responsive layout (desktop & mobile)
- 🌙 Dark mode toggle
- 🍪 Mobile hamburger menu
- 🛒 Product cards with pricing and order buttons

## Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (DOM manipulation)
- Font Awesome icons
